<h2><b>PowerCo SME Churn Prediction</b></h2>
<b>Project Overview</b>

This project analyzes SME customer churn for PowerCo to determine whether price sensitivity drives churn and to prepare data for predictive modeling.

<b>Objective</b>

- Identify key drivers of churn
- Engineer business-relevant features
- Prepare dataset for churn prediction model
- Provide actionable retention recommendations

<b>Key Insights</b>

- Pricing and tenure-related variables show strong correlation with churn.
- High price exposure and shorter tenure increase churn risk.
- Feature engineering improved predictive readiness of the dataset.

<b>Business Impact</b>

- A churn prediction model would allow PowerCo to:
- Identify high-risk customers early
- Apply targeted retention strategies
- Reduce revenue loss from SME churn

<b>Tools Used</b>

- Python 
- Pandas 
- NumPy 
- Matplotlib 
- Jupyter Notebook
